# powercfg_generator

## 简介
这个工具用于生成 `powercfg` 脚本和卡刷包
生成的卡刷包支持自动识别平台并安装对应调度

## 许可证
本工具使用 GPL v3 许可证发布

所有使用本工具生成的 `powercfg` 脚本和卡刷包也应遵循 GPL v3 许可证

## 需要的环境
bash + zip

## 使用方法
1. clone 这个repo
2. 执行 start.sh
3. 新建一个项目（id不能包含空格）
